# Configuration
---

To fully use Hyper Megaphone you will need to configure the script. Below is an example config and below that is a table explaining what each option does.

```js
{
    "name": "BanditRP",
    "discordURL": "https://discord.gg/BanditRPAus",
    "imageURL": "https://i.imgur.com/8Q9QX9m.png"
}
```

| Name        | Description |   Type |
| :------------ | :---------: | ----------: |
| name      |    The name that appears when a player is rejected    | `String` |
| discordURL |    The discord invite link to your server.      |    `String` |
| imageURL | The image that appears when a player is rejected | `String` |
