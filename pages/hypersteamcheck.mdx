# Hyper SteamCheck

Hyper SteamCheck is a Fivem resource that enhances your server's experience by checking if users have Steam running. If a user doesn't have Steam running, the bot provides them with a prompt to open Steam.

## Features

- Seamless integration: Easily add Hyper SteamCheck to your Fivem Server and enhance user engagement.
- Steam status verification: Automatically checks if users have Steam running.
- User-friendly prompts: Users without Steam running receive a friendly prompt to open Steam.
