# Configuration
---

To fully use Hyper Megaphone you will need to configure the script. Below is an example config and below that is a table explaining what each option does.

```lua
Config = {}

Config.range = 50.0 -- The range of the megaphone in meters.
Config.DefaultKeybind = 'L'
```

| Name        | Description |   Type |
| :------------ | :---------: | ----------: |
| range      |    The range in meters that the megaphone reaches    | `Int` |
| DefaultKeybind |    The default keybind in the fivem menu. (Configurable in keybinds)     |    `string` |
